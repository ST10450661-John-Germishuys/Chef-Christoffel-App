# Chef Christoffel App

## Overview

Chef Christoffel App is a React Native mobile application developed using Expo and TypeScript. The application allows chefs to add menu items, categorise dishes, display menu information, and calculate total menu items and prices.

## Features

- Add menu items
- Enter dish descriptions
- Select course categories
- Display menu list
- Calculate total menu items
- Calculate total menu price
- Delete menu items
- Input validation
- Responsive mobile design

## Technologies Used

- React Native
- Expo
- TypeScript
- JavaScript
- VS Code
- GitHub

## Installation

Run the following commands:

```bash
npm install
npx expo start
```
